## Usage

Clone this repository
```sh
$ git clone https://github.com/miya/to-do
```

Move to directory
```sh
$ cd to-do/
```

Install dependencies
```sh
$ yarn install
```

Runs the app in the development mode
```sh
$ yarn start
```
